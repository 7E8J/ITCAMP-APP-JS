1. Variable and data types

Variable var, let, const

Example: 

      var x = 5;
      var y = 6;
      var z = x + y;
      //result z = 11

      let x = 5;
      let y = 6;
      let z = x + y;
      //result z = 11

      const price1 = 5;
      const price2 = 7;
      const total = price1 + price2;
      total = 11111
      //result TypeError: Assignment to constant variable.


tip: When to Use JavaScript var? Always declare JavaScript variables with var,let, orconst. The var keyword is used in all JavaScript code from 1995 to 2015. The let and const keywords were added to JavaScript in 2015. If you want your code to run in older browsers, you must use var.

Blockscope

Example: 

      var x = 10
      if(x == 10){
        var y = x
        console.log("y in " + y)
      }
      console.log("y out " + y)


Example 2: 

      var x = 10
      if(x == 10){
        let y = x
        console.log("y in " + y)
      }
      console.log("y out " + y)


Example 3: 

      let x = 10
      if(x == 10){
        let x = 11
        console.log("x in " + x)
      }
      console.log("x out " + x)
