5. Array & Object

Array

    let array_name = [item1, item2, ...];

Example:

  const cars = [
    "Saab",
    "Volvo",
    "BMW"
  ];

Accessing Array Elements

    array[index]

Example:

  cars[1] // Saab
  cars[2]  //Volvo
  cars[3]  //BMW

Array Properties and Methods

  cars.length   // Returns the number of elements
  cars.sort()   // Sorts the array

Example:

  const fruits = ["Banana", "Orange", "Apple", "Mango"];
  let length = fruits.length;

Object

Property Property Value
firstName "โจนาธาน"
lastName "โจสตาร์"
id 65070001
fullName function() {return this.firstName + " " + this.lastName;}

Example:

  const student = {
    firstName:"โจนาธาน", 
    lastName:"โจสตาร์", 
    id:"65070001",
    fullName: function() {
      return this.firstName + " " + this.lastName;
    },
    course: ["Math","Eng","Com"]
  };

Accessing Object Methods

  objectName.methodName()

Example:

  student.firstName // โจนาธาน
  student.lastName // โจสตาร์
  student.id // 65070001
  student.fullName // [Function: fullName]
  studnet.course // ["Math","Eng","Com"]
